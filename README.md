# thorcon
Thor Container Runtime (**thorcon**) is a container runtime written in Zig

thorcon is an implementation of the [OCI runtime-spec](https://github.com/opencontainers/runtime-spec) in Zig, similar to [crun](https://github.com/containers/crun), [runc](https://github.com/opencontainers/runc) and [youki](https://github.com/youki-dev/youki).

## Requirements

Zig version >= 0.13

## Build

```
$ make build
```
